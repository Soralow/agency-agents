## 🚀 How to Use This Registry

### Method 1: Direct Activation
```
In VS Code Copilot Chat:
"Activate the Frontend Developer agent"
"Switch to Backend Architect mode"
"I need the Code Reviewer for this PR"
```

### Method 2: Browse This File
1. Search this file for your task
2. Click the agent link or note the path
3. Open the agent's `.md` file
4. Copy the content into Copilot Chat

### Method 3: Use the Shell
```bash
# Find an agent by name
grep -r "^name:" * | grep "Frontend"

# View an agent
cat engineering/engineering-frontend-developer.md
```

## 💡 Pro Tips

### Combination Recipes

**Build a Feature End-to-End:**
1. Product Manager → Define requirements
2. Backend Architect → Design API & database
3. Frontend Developer → Build UI
4. Code Reviewer → Review code
5. DevOps Automator → Deploy

**Launch a Product:**
1. Product Manager → Define scope
2. Go-to-Market Strategist → Plan launch
3. Growth Marketer → Create marketing
4. Sales Development Rep → Enable sales
5. Customer Success Manager → Support customers

**Handle Security Incident:**
1. Incident Response Commander → Lead response
2. Security Architect → Assess impact
3. Senior Developer → Fix vulnerability
4. Application Security Engineer → Prevent recurrence

**Optimize Performance:**
1. Backend Architect → Identify bottleneck
2. Database Optimizer → Optimize queries
3. Performance Tester → Validate improvements
4. DevOps Automator → Monitor in production
