## ⚡ Speed Tips

**Fastest activation:**
- Type just the agent's first name or specialty
- "Frontend?" instead of full name
- "DB?" for Database Optimizer
- "DevOps?" for DevOps Automator

**Find agent quickly:**
- Search AGENTS.md for your task
- Use Quick Finder in AGENTS_QUICK_REFERENCE.md
- Ask "Which agent for X?"
