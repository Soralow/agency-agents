   - Frontend Developer → UI Designer → Code Reviewer → DevOps Automator

   - Building a feature: Product Manager → Backend Architect → Frontend Developer → QA Engineer
   - Launching a product: Product Manager → Growth Marketer → Sales Dev Rep → Customer Success Manager
   - Security incident: Incident Response Commander → Security Architect → Senior Developer

3. **Agent rotation by phase**
   - Planning: Product Manager, Strategist agents
   - Building: Engineering agents
   - Testing: QA, Security, Performance agents
   - Launching: Marketing, Sales agents
   - Supporting: Customer Success agents
