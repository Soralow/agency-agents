## 🚀 Setup Instructions

### Quick Deploy to Your Tools

**For GitHub Copilot / VS Code:**

```bash
# Navigate to your agency-agents repo
cd /Users/benjaminlow/Documents/GitHub/agency-agents

# Run the install script (interactive wizard)
./scripts/install.sh

# Or install specific divisions
./scripts/install.sh --tool copilot --division engineering,design,security
```

**For Other AI Tools (Cursor, Windsurf, Aider, OpenCode, etc.):**

```bash
# Convert to supported formats (optional step)
./scripts/convert.sh

# Install to your preferred tool
./scripts/install.sh --tool cursor
./scripts/install.sh --tool windsurf
./scripts/install.sh --tool aider
```

### Manual Setup

**Preferred way to use these:** Copy the `.md` file of any agent you want directly into your prompt/instruction area:

1. Browse to `/Users/benjaminlow/Documents/GitHub/agency-agents/{division}/`
2. Copy the agent file content (e.g., `engineering-frontend-developer.md`)
3. Paste into your AI tool's instruction or prompt field
4. Activate: "Use the Frontend Developer agent" or "Switch to Frontend Developer mode"

## 💡 Pro Tips

### 1. **Pick the Right Agent for the Task**
Each agent has a specific expertise. Don't use the "Senior Developer" for UX design — use the "UX Architect" instead. Precision = better results.

### 2. **Combine Agents for Complex Tasks**
Need to build, design, AND secure an app? Chain three agents:
```
1. UX Architect → Design the system
2. Backend Architect → Plan infrastructure
3. Application Security Engineer → Add security
```

### 3. **Use Agent Personalities**
Agents have unique communication styles. The "Rapid Prototyper" is fast and iterative. The "Software Architect" is methodical. Pick the personality that fits your work style.

### 4. **Create Custom Combos**
Bookmark or reference these combinations for your workflows:

- **Building a React app** → Frontend Developer + UI Designer + Code Reviewer
- **Launching a product** → Product Manager + Growth Marketer + Sales Development Rep
- **Securing infrastructure** → Security Architect + DevOps Automator + Incident Response Commander
- **Writing technical docs** → Technical Writer + Frontend Developer + Prompt Engineer

### 5. **Rotate Based on Phase**
Different phases need different agents:
- **Planning** → Product Manager, Strategy agents
- **Building** → Engineering agents
- **Testing** → QA, Security Tester
- **Launch** → Go-to-Market Strategist, Growth Marketer
- **Post-Launch** → Customer Success Manager, Product Manager

### 6. **Agent Synergy**
Certain agents work well together. Example:
- Marketing Automation Engineer + Email Intelligence Engineer (email campaigns)
- Database Optimizer + Backend Architect (query optimization)
- UX Researcher + Behavioral Nudge Engine (user engagement)
